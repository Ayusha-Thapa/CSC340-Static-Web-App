About Us

This website is a character gallery inspired by the Harry Potter series, which I chose because it is a well-known fictional universe with a wide range of memorable characters and clearly defined traits. The series features characters with distinct personalities, values, and roles, making it a strong fit for a character-based website.

The Harry Potter world allows characters to be meaningfully categorized in multiple ways, including Hogwarts houses based on personality and values (such as Gryffindor, Hufflepuff, Ravenclaw, and Slytherin), blood status, and moral or organizational allegiance. These built-in categories made it easier to design a structured and searchable character gallery.

When designing the website, I focused on keeping the layout simple and visually organized using Bootstrap components like the navigation bar, grid system, and cards. Overall, this project allowed me to combine a popular fictional universe with basic web development concepts to create an interactive and easy-to-use character gallery.
