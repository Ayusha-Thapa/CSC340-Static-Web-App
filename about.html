About Us

This website is a character gallery inspired by the Harry Potter series. I chose the Harry Potter universe because it is a fictional world that many people are familiar with and enjoy. The series features a wide range of characters, each with unique roles, personalities, and abilities, which made it a good fit for a character-based website.

Another reason I chose this theme is because the characters can be easily categorized by things like house, role, and skills. This made it easier to design a search feature that allows users to filter characters in a meaningful way. The clear structure of the Harry Potter world helped guide my design choices and layout.

When designing the website, I focused on keeping the layout simple and visually organized. I used Bootstrap components such as the navigation bar, grid system, and cards to create a clean and responsive design. The goal was to make the site easy to navigate while still being visually appealing and consistent with the theme.

Overall, this project allowed me to combine a well-known fictional universe with basic web development concepts to create an interactive and easy-to-use character gallery.
